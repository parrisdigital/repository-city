# Contributing

Thanks for helping improve Repository City.

1. Fork the repository and create a focused branch such as `feat/camera-controls` or `fix/tree-filtering`.
2. Install dependencies with `pnpm install`.
3. Copy `.env.example` to `.env.local`; a GitHub token is optional for public repositories.
4. Run `pnpm dev` while developing.
5. Run `pnpm check` before opening a pull request.
6. Include screenshots or a short recording for visible changes.

Keep changes small, preserve accessibility, and avoid adding dependencies when a short local implementation is clearer.
